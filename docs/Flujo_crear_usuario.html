1. CREACIÓN DE USUARIOS PARA EL SISTEMA

Para poder crear usaurios en el sistema se sigue el siguiente flujo de trabajo:

Primero se ingresa al modulo de personal como se muestra a continuación

2.

Para poder registrar un personal, se debe hacer click en el botón añadir

3. Datos del personal

Aparecerá un formulario con los datos que se requiere guardar del personal

4.

Se completan los datos del personal correspondiente:

5.

Una vez completa la información se procede hacer click en guardar para poder registrar al personal

6.

Si la información proporcionada es correcta les aparecerá un mensaje como se muestra a continuación, caso contrario aparecerá un mensaje de error indicando el problema que sucedió.

7.

Luego de registrar el personal automáticamente aparecerá en la lista

8. Editar personal

Para editar los datos del personal se procederá hacer click en el siguiente botón

9.

A lo que el sistema mostrara una ventana con los datos del personal que se quiere edita como se muestra en la imagen, haciendo todos los cambios correspondientes se procede hacer click en el botón guardar para poder completar la edición.

10. Eliminar personal

Para eliminar un registro de un personal se procederá hacer click en el botón que se muestra a continuación

11.

Al momento de hacer click se les mostrara un mensaje como el que aparece en pantalla para confirmar o denegar la acción:

12. Crear usuario

Una vez creado el personal, se procede a dirigirse a al modulo de usuario

13. Vista del modulo usuarios

14.

Se procede hacer click en el botón añadir para procede con la creación del usuario

15. Click on dialog

Se mostrara el siguiente formulario para el registro de los datos

16.

Una vez completada toda la información necesaria se procederá con el registro

17.

Si todo es correcto se mostrará un mensaje como el que se muestra a continuación

18.

Automáticamente aparecerá el usuario recientemente registrado en la vista de usuarios

19. Editar usuario

Para editar un usuario se debe de hacer el siguiente botón que se indica

20.

Al momento de hacer click se abrirá una ventana con los datos del usuario, donde podrá editar su usuario y CREAR una nueva contraseña en caso no la recuerde

21.

Realizados los cambios correspondiente se procedera a guardar los datos

22.

Les aparecerá un cuadro de confirmación para validar si se realiza o no la acción

23. Eliminar usuario

Para eliminar un usuario, debe realizar click al botón que se señala a continuación

24.

Automáticamente aparecerá un cuadro para verificar si esta seguro de realizar la acción, según sea el caso se selecciona una y termina el proceso de registro de usuario

25. Asignar roles

Para poder asignar roles al sistema , el personal debe de contar con un usuario previamente registrado en el sistema, caso contrario no se podrá realizar el registro

26. Vista de asignar roles

27.

Para asignar un rol a un usuario se debe hacer click en añadir

28.

Donde se mostrar el siguiente cuadro, se tendrá que completar con el DNI del usuario y seleccionar el rol que se le quiera dar

29.

Si se completo la información se procederá a guardar

30.

Si el registro se realizó correctamente aparecerá el siguiente mensaje, sino se mostrará los errores que sucedieron

31.

Automáticamente se listara el registro del rol a usuario que se creo anteriormente

32. Editar rol

33.

Cargara un cuadro con los datos del usuario

34.

Luego de realizar los cambios correspondientes, se procede a guardar la información

35.

Aparece un cuadro para confirmar o denegar la acción

36.

Luego de confirmar la acción, aparecerá el siguiente mensaje de confirmación

37. Eliminar Rol del usuario

Para eliminar un rol de un usuario, se hace click en el botón que se muestra a continuación

38.

Automáticamente se muestra el siguiente cuadro donde se podrá confirmar o denegar la acción

39. Fin del Flujo para crear un usuario
